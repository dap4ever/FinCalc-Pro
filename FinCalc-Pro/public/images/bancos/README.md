# Logos dos Bancos

Adicione os logos dos bancos nesta pasta com os seguintes nomes:

- **caixa.png** - Logo da Caixa Econômica Federal
- **bb.png** - Logo do Banco do Brasil
- **itau.png** - Logo do Itaú
- **santander.png** - Logo do Santander
- **bradesco.png** - Logo do Bradesco

## Recomendações:
- Formato: PNG com fundo transparente
- Tamanho recomendado: 200x80px (proporção 2.5:1)
- Peso máximo: 50KB por arquivo
- Qualidade: Alta resolução para telas Retina

## Onde encontrar:
- Sites oficiais dos bancos (seção de imprensa/mídia)
- [Brandfetch](https://brandfetch.com) - Busque pelo nome do banco
- [Worldvectorlogo](https://worldvectorlogo.com) - Logos em vetor

Após adicionar os logos, eles aparecerão automaticamente no dropdown de seleção de bancos!
